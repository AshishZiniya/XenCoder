# FindGarden V2

FindGarden V2 is a high-contrast, node-based canvas for AI-assisted thinking and organized note-taking. Built with a focus on speed, clarity, and premium aesthetics, it provides a powerful workspace for developers and gardeners of ideas.

Made by **Ashish Zinya**.

## 🌟 Key Features

### 🖼️ Node-Based Canvas
- **Infinite Workspace**: Organize your thoughts, chats, and notes on an expansive canvas powered by `@xyflow/react`.
- **Dynamic Interaction**: Easily create, move, and connect nodes to map out complex topics.
- **Smart Drag & Drop**: Fluid and responsive node management.

### 🤖 AI-Powered Insights
- **Cerebras Intelligence**: Integrated with Cerebras Cloud SDK for lightning-fast AI responses.
- **Contextual Ask/Explain**: Select any text within a chat to "Explain" it or "Ask" a follow-up question, spawning new contextual nodes.
- **System Instructions**: Customize how the AI behaves with dedicated system instruction controls.

### ✒️ Rich Markdown Support
- **Vibrant Syntax Highlighting**: Clean, readable code blocks with "One Dark Pro" inspired aesthetics.
- **Mathematical Precision**: Full support for $LaTeX$ and complex formulas via `remark-math` and `rehype-katex`.
- **GFM Ready**: Standard GitHub Flavored Markdown for tables, lists, and more.

### 🖤 Premium High-Contrast UI
- **Permanent Dark Theme**: A sleek, definitive black and white aesthetic for maximum focus and reduced eye strain.
- **Barlow Typography**: Modern, crisp font choice standardized across the entire application.
- **Optimized Readability**: Carefully tuned font sizes and spacing for a superior reading experience.

### 🛠️ Advanced Controls
- **History Management**: Full Undo/Redo support to navigate your creative journey.
- **Responsive Zoom**: Seamlessly navigate large canvases with dedicated zoom and "Fit View" controls.

## 🚀 Tech Stack

- **Framework**: [Next.js 16](https://nextjs.org/) (App Router)
- **Runtime**: React 19
- **Typography**: [Barlow](https://fonts.google.com/specimen/Barlow)
- **Styling**: Tailwind CSS 4 & Vanilla CSS
- **State Management**: [Jotai](https://jotai.org/)
- **Node Engine**: [@xyflow/react](https://reactflow.dev/)
- **AI SDK**: Cerebras Cloud SDK
- **Markdown Processing**: `react-markdown`, `remark-gfm`, `rehype-highlight`

## 🏁 Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
